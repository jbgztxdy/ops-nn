## 功能说明

- 算子功能：实现张量的 Lp 范数计算与归一化功能，支持对输入张量按指定维度计算 p 范数，并使用对应范数对输入进行归一化。
- 支持全局、按行、按列三种计算模式。

- 计算公式：

设输入张量为 \(x\)，范数阶数为 \(p\)，数值稳定项为 \(\epsilon\)。

- **Lp 范数计算：**

$$
\text{norm} = \left( \sum |x_i|^p + \epsilon \right)^{1/p}
$$

- **归一化结果：**
$$

y_i = \frac{x_i}{\text{norm}}

$$
其中，范数的计算范围由 `axis` 参数决定。

## 参数说明

| 参数名 | 输入/输出/属性 | 描述 | 数据类型 | 数据格式 |
|------|---------------|------|---------|---------|
| x | 输入 | 待进行 LpNormV2 计算的输入张量 | FLOAT、FLOAT16 | ND |
| p | 属性 | Lp 范数的阶数 | FLOAT | - |
| axis | 属性 | 范数计算维度：0（全局），1（列），2（行） | INT | - |
| epsilon | 属性 | 数值稳定补偿项 | FLOAT | - |
| y | 输出 | 归一化后的输出张量 | FLOAT、FLOAT16 | ND |
