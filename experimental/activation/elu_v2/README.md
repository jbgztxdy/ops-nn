## 功能说明

- 算子功能：计算指数线性单元激活函数Elu的结果。
- 计算公式：

$$
f(x) =
\begin{cases} 
\text{scale} \cdot x , \quad x > 0 \\ 
\text{scale} \cdot \alpha \cdot (e^{\text{input\_scale} \cdot x} - 1) , \quad x \leq 0 
\end{cases}

$$

## 参数说明

| 参数名         | 输入/输出/属性 | 描述                    | 数据类型           | 数据格式 |
| ----------- |:-------- | --------------------- | -------------- | ---- |
| x           | 输入       | 输入x                   | fp16、fp32、bf16 | ND   |
| alpha       | 输入       | 参数alpha，默认值为1.0       | fp32           | (1,) |
| scale       | 输入       | 参数scale，默认值为1.0       | fp32           | (1,) |
| input_scale | 输入       | 参数input_scale，默认值为1.0 | fp32           | (1,) |
| y           | 输出       | 输出y                   | fp16、fp32、bf16 | 同输入x |
