## 功能说明
- 算子功能：对输入tensor逐元素计算修正线性单元，输出计算结果，并同时返回一个掩码用于指示输入元素中哪些值大于0。
- 计算公式：
    $$y_i = \max(x_i, 0)$$
    $$mask_i = (x_i > 0)$$
## 参数说明
| 参数名  | 输入/输出/属性 | 描述                            | 数据类型                                    | 数据格式    |
|------|----------|-------------------------------|-----------------------------------------|---------|
| x    | 输入       | 待进行relu_v3计算的入参，公式中的$x_i$。    | FLOAT16,FLOAT,INT32,INT8,UINT8,BFLOAT16 | NC1HWC0 |
| y    | 输出       | 待进行relu_v3计算的出参，公式中的$y_i$。    | FLOAT16,FLOAT,INT32,INT8,UINT8,BFLOAT16 | NC1HWC0 |
| mask | 输出       | 待进行relu_v3计算的出参，公式中的$mask_i$。 | UINT8                                   | ND      |
## 约束说明
x的最后一维必须是8的倍数。
